# 查询语言

本部分介绍InfluxQL，InfluxDB的SQL类查询语言，用于查询db中的数据。

## InfluxQL教程
本部分的前七个文档提供了InfluxQL的教程式介绍。你可以随时下载文档相关的示例数据。

### [schema查询语法](schema_exploration.md)

涵盖InfluxQL用于管理InfluxDB中的数据库和存储策略，具体有创建删除数据库和存储策略，以及删除数据。
